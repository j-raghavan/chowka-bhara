# public/ assets

Drop a photo named **`board-reference.jpg`** in this folder to feature the
real handmade board on the home screen. The home screen already renders a
CSS replica of that board (lime-green safe houses with green ×, blue pawn
glyphs on the start houses, and the crown medallion in the center), so the
photo is optional — if present it appears beneath the replica, if absent the
replica stands on its own.

Files in `public/` are copied to the site root at build time and are served
under the GitHub Pages base path automatically.
